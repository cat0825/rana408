## 项目简介

本项目专为考研学子打造，收集和整理了 408 考研科目中的**启发性问题**，旨在帮助同学们开阔思路、发现知识盲区、提升理解深度。内容涵盖操作系统、计算机组成原理、数据结构、计算机网络等核心模块。

> ⚠️ 本项目**不是**一份全面的复习指南，也不包含所有基础知识点。请结合权威教材和系统课程进行复习，本项目内容仅供查漏补缺和思维训练参考。

## 目录结构

- `408/`：各科目启发性问题与知识点笔记
- `notes/`：补充笔记与个人总结
- `数学记录/`：数学相关的考研资料
- `附件/`：配套图片、资料等资源
